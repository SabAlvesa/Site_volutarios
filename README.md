# Sistema de Cadastro de Voluntários

Este projeto é uma aplicação web simples para cadastro, listagem e gerenciamento de voluntários. 

## Funcionalidades

- Cadastro de voluntários com dados pessoais e endereço, incluindo consulta automática de endereço pelo CEP usando a API ViaCEP.
- Login e controle de acesso baseado em dados armazenados no LocalStorage do navegador.
- Exibição de lista de voluntários cadastrados, com filtro em tempo real pelo nome.
- Opção de remover voluntários individualmente ou limpar toda a lista.
- Excluir o próprio cadastro com confirmação.
- Interface organizada em abas para facilitar a navegação entre cadastro, lista e dados pessoais.
- Imagem de perfil gerada aleatoriamente para cada usuário.

## Tecnologias utilizadas

- HTML, CSS e JavaScript puro 
- API ViaCEP para consulta automática de endereço pelo CEP
- LocalStorage para persistência de dados no navegador
